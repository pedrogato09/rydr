--------------------------------------------------------------------------------
ABOUT WEBP FORMAT
--------------------------------------------------------------------------------

WebP is a modern image format developed by Google that provides superior
compression for images on the web. Key benefits include:

  - 25-34% smaller file sizes compared to PNG
  - Supports transparency (alpha channel)
  - Both lossy and lossless compression modes
  - Supported by all modern browsers (Chrome, Firefox, Safari, Edge)

--------------------------------------------------------------------------------
USING YOUR WEBP FILES
--------------------------------------------------------------------------------

WebP files work great for:

  - Websites and web applications (faster load times)
  - Email attachments (smaller file sizes)
  - Social media (most platforms support WebP)
  - Digital archives (lossless mode preserves quality)

HTML Example:
  <img src="your-image.webp" alt="Description">

With Fallback:
  <picture>
    <source srcset="image.webp" type="image/webp">
    <img src="image.png" alt="Description">
  </picture>
